ویجت پیشنهاد فریم عینک

این صفحه نمونه‌ای از ویجت پیشنهاد فریم عینک است. دکمه شناور سمت راست صفحه را مشاهده کنید و روی آن کلیک کنید تا ویجت باز شود. با استفاده از این ویجت، کاربران می‌توانند فریم‌های مناسب با شکل صورت خود را پیدا کنند.

نکته: در این نسخه نمونه، نیازی به تنظیم آدرس API نیست و از داده‌های نمونه استفاده می‌شود.

این یک محتوای نمونه است

فرض کنیم این یک وب‌سایت فروشگاهی است که فروشنده عینک می‌باشد. کاربر در حین مرور محصولات این سایت، می‌تواند با کلیک روی دکمه شناور، ویجت پیشنهاد فریم عینک را باز کند و فریم‌های متناسب با شکل صورت خود را مشاهده کند.

با استفاده از این ویجت، کاربران می‌توانند تصویر صورت خود را آپلود کنند یا از دوربین استفاده کنند. سپس ویجت شکل صورت آن‌ها را تشخیص داده و فریم‌های مناسب را پیشنهاد می‌دهد.

این ویجت کاملاً مستقل است و می‌تواند به هر وب‌سایتی اضافه شود.

نحوه استفاده در وب‌سایت شما

برای استفاده از این ویجت در وب‌سایت خود، کافی است اسکریپت زیر را در صفحه HTML خود قرار دهید:

<script src="http://your-server.com/eyeglass-widget.js"></script> <script> new EyeglassWidget({ floatingButton: true, position: 'left', // یا 'right' buttonText: 'پیشنهاد فریم عینک' }); </script>

در صورتی که می‌خواهید به API خودتان متصل شوید، می‌توانید آدرس API را نیز تنظیم کنید:

<script src="http://your-server.com/eyeglass-widget.js"></script> <script> new EyeglassWidget({ apiEndpoint: 'https://your-api.com/api/v1/analyze', floatingButton: true, position: 'left', buttonText: 'پیشنهاد فریم عینک' }); </script>
